# WaterWithFFT
by Gabriel Nkole

## Description
Ocean rendering on a 1000x1000 plane using the Phillips Spectra & an optimized GPU FFT algorithm.

## Features
- Choppy Waves
- Scatter Lighting
- PBR Specular Reflections
- Image-based Lighting (IBL)
- Sea Foam
